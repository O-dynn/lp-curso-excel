# 🟣 Landing Page — Curso Intensivo de Excel

Landing Page desenvolvida para um curso intensivo (fictício) de Excel, com foco em **conversão e performance**.

## 🎯 Objetivo
Gerar cadastros e conversões para um curso intensivo de Excel, com uma experiência simples, rápida e persuasiva.

---

## 💡 Destaques do Projeto
- ✍️ **Copywriting estratégico** com foco em ação e benefícios reais.  
- 🎨 **Design limpo e responsivo**, adaptado para mobile e desktop.  
- 🚀 **SEO otimizado**: títulos, descrições e estrutura HTML semântica.  
- ⚡ **Formulário funcional** com integração ao [FormSubmit](https://formsubmit.co/).
